# Notice for Publication 20260515.1

Publication: Gaia-HIP supplemental display de-duplication map

Release: `20260515.1`

This publication contains a Found-in-Space supplemental mapping delta intended
to reduce visible duplicate artefacts when rendering Gaia and Hipparcos stars
together. It does not republish the official Gaia-Hipparcos2 best-neighbour
crossmatch table.

## Upstream Data Sources

This publication depends on, compares against, or derives evidence from:

- ESA Gaia DR3 `gaiadr3.gaia_source`
- ESA Gaia DR3 `gaiadr3.hipparcos2_best_neighbour`
- ESA Gaia DR3 `gaiadr3.hipparcos2_neighbourhood`
- Hipparcos-2 / "Hipparcos, the New Reduction" catalogue, VizieR catalogue
  `I/311`
- VizieR catalogue access service, CDS, Strasbourg, France
